# suga_keiba_projects
競馬予想webアプリです。
suga_keiba_projectsからスクレイピング、データ形成して、
作成した予想図をホームのフォームから検索し、表示します。
また、掲示板、ブログ機能を搭載しています。
 
# 使用環境、ライブラリ
* Python                    3.9.6
* beautifulsoup4            4.9.3
* matplotlib                3.4.1
* pandas                    1.2.4
* numpy                     1.20.2
* ttkthemes                 3.2.2
